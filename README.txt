Project: TestNGExample

Description: Project contains sample scripts and framework set up to develop and explore Web Automation testing strategies.

Step to Run from Command Prompt:
1. Naviagate to root directory of the project
2. Open Command Prompt for the root location
3. Configure TestSuite.xml to run test scripts.
4. Execute Maven command to execute the scripts. (mvn test)
5. It will download the dependencies and other required files.
6. It will execute configured test cases.
